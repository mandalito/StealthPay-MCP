Check your wallet balances.

Use the stealthpay `get-balances` MCP tool to show the user's ETH and token balances.

If the user specifies a chain, pass it. Otherwise use the default.

After showing balances, if the user seems to want to send funds, suggest using `/stealthpay-send`.

Arguments: $ARGUMENTS
The optional argument is a chain name (e.g., "sepolia", "base", "ethereum").
