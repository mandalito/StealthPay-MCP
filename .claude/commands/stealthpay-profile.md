Look up a payment profile for an ENS name.

Use the stealthpay `get-payment-profile` MCP tool to resolve the ENS name and show:
- Resolved address
- Avatar and description (if set)
- Whether stealth keys are registered
- Stealth meta-address (if available)

Arguments: $ARGUMENTS
The argument should be an ENS name (e.g., "vitalik.eth").
